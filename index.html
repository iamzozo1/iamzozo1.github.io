Arcade Project

Arcade is a gaming platform: a program that lets the user choose a game to play and keeps a register of player scores.

To be able to deal with the elements of your gaming platform at run-time, your graphics libraries and your games must be implemented as dynamic libraries, loaded at run-time. Each GUI available for the program must be used as a shared library that will be loaded and used dynamically by the main program. This also applies to your games.

Adding a New Library

To add a new library, there is no particular rule to implement it as long as the required interfaces are used. The new libraries should be added to their corresponding folders in lib (either graphics or games), and they will be automatically detected by the Arcade program.
