void arcade::Core::run ( std::string  defaultGraphLib)

Main execution loop for the Core class.

This function initializes the list of available games and graphic libraries by scanning their respective directories. It then loads the default graphic library and sets up the menu as the initial game. The function enters a loop that continues running as long as _isRunning is true.

Within the loop:

  • It polls events from the current display module.
  • Handles the input keys using the handleKeys function.
  • Updates the current game state based on the input keys.
  • Retrieves and processes the map entities from the current game.
  • If the menu game is active and a game is selected, it validates the selected game's file path.

If the selected game file does not exist, an error is logged, and the game selection is reset.
